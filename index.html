🔗 PrimeMLS Connection
This connects your live MLS listings to Broker OS. You need two things: your proxy server URL (from Railway/Render — free) and your PrimeMLS credentials. See the steps below.
Step 1 Deploy the proxy — Upload the broker-proxy folder to GitHub, then connect it to Railway or Render (free). They give you a URL like https://broker-proxy-xxx.up.railway.app

Step 2 Get MLS credentials — Log into PrimeMLS → Member Tools → IDX/API Access. If no API tab, call them at (603) 930-3220 and ask for RESO Web API credentials.

Step 3 Paste both below and hit Save & Test.
